*Last updated: 2026-06-17*

## Returns

- don't document `<returns>` by default — the `<summary>` states what's returned (`Gets …`, `Creates …`)
- add `<returns>` only when the return carries detail the summary can't (e.g. a tri-state outcome, a nullable's meaning)
