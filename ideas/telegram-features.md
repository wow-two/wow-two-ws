# Telegram — Features Catalog

*Last updated: 2026-05-12*

> Compact reference of all Telegram features (core + 2025-2026 updates), categorized.
> Source for ideation around Telegram bot / SDK / mini-app projects in the wow-two ecosystem.

## Messaging

- Text · voice messages · video messages · stickers · GIFs · emoji · custom emoji
- Replies · quotes · reactions · forwards · pin messages · scheduled send · silent send · self-destruct timers
- Edit/delete (any time, both sides) · drafts sync · saved messages
- Polls · quizzes · checklists · live locations
- Multi-device sync (cloud chats) · message search · jump-to-date
- Translation (per-message + chat-wide)

## Calls

- 1:1 voice · video · screen share · noise suppression
- E2EE (calls always; secret chats opt-in)

## Groups

- Up to `200K` members · admin roles + permissions · topics (forum mode) · slow mode
- Voice chats / live audio · video chats · group calls
- Member tags (roles/interests) · join requests · invite links w/ expiry
- Reply boards · anti-spam · group ownership transfer

## Channels

- Unlimited subscribers · post as channel · stories as channel
- Boosts (from Premium subs) → unlock custom emoji, wallpapers, stories, profile color
- Discussion groups (channel ↔ group link) · sponsored messages
- Public/private · join-by-request · Telegram Stars monetization

## Stories

- Post from profile / channel · dual-camera · edit after posting · rewind/fast-forward
- Privacy lists · custom durations (`6/12/24/48h`, Premium) · stealth view (Premium)
- Permanent view history (Premium) · save others' stories (Premium) · longer captions w/ links

## Files & Media

- Up to `2 GB`/file (`4 GB` Premium) · unlimited cloud storage · any file type
- Photo/video editor · spoiler media · view-once · live photos
- GIF captions · sticker/emoji packs · 100M+ sticker/emoji search

## Bots & API

- Open Bot API · inline bots · mini apps (web apps) · payments · games
- `@mention` bots in any chat (no-membership) · bot-to-bot chats · bots managed by bots
- Guest AI bots · custom AI styles · colored bot buttons

## Privacy & Security

- Cloud chats (server-client E2E) · Secret Chats (full E2EE, device-only)
- Granular privacy: last seen · phone # · profile photo · forwards · calls · voice msgs · bio · birthday · gifts
- 2FA · passkeys (no SMS) · login alerts · active sessions
- Block/report · anonymous group admins · "Who can see my number"
- Disable sharing in private chats — blocks forward/copy/screenshot/save (Premium)
- Hidden number (sign in via username only)

## Customization

- Themes · chat wallpapers (per-chat) · animated backgrounds · color schemes
- Folders (auto-fill rules · unread folder) · app icons
- Profile: name color · animated emoji status · bio link · birthday · profile video · music in profile
- Username (`@`) · public link · multi-username · usernames marketplace (Fragment)

## Telegram Premium

- Ad-free · `4 GB` uploads · 4× pinned chats / folders · 2× channel subs · 1000 chats/folder
- Voice→text transcription · real-time chat translation
- Premium stickers / reactions / emoji statuses
- Faster downloads · advanced chat management · profile badge
- Story perks (above) · Grok AI assistant (xAI) · last-seen "no read receipt"
- Boosts to give channels

## Telegram Business

- Business hours · location · greeting/away messages · quick replies
- Chatbots for business · custom intro · deep links · Telegram for Business app

## TON / Stars / Gifts

- Telegram Stars (in-app currency) · Stars subscriptions · paid posts/media
- Gifts (NFT-style collectible) · gift crafting · gift auctions · scam protection on gift purchases
- Fragment marketplace (usernames, numbers, gifts)
- TON wallet integration

## Recent Updates — 2025

- 13 major releases · 75+ features · ~26-day cadence
- Collectible gifts (combinable) · member tags · disable-sharing · GIF captions · Grok integration
