## Adding a new entry

1. Drop the file under the right subfolder (`platforms/`, `dbs/`, etc.) — create the subfolder if it's a new category
2. Use the file shape: **What it is · Core products · What devs can build · Why pick it · Links**
3. Add a row to the table above
4. Keep it compact — link out to vendor docs instead of restating them
